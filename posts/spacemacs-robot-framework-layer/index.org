This week support for the [[https://github.com/tomi/vscode-rf-language-server][Robot Framework Intellisense]] language server was added
to Emacs's lsp-mode.

For easy installation a spacemacs layer that also uses the
[[https://github.com/dougm/robot-mode][robot-mode package]] by Doug MacEachern can be found [[https://github.com/dalanicolai/robot-framework-layer][here]].
